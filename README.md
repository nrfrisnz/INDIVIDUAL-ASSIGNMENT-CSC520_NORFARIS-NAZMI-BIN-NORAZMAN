## Scripts Included

### 1. system_info.sh

This script displays system information and saves the output into a log file.

Main functions:

* Displays welcome message
* Shows current date and time
* Shows disk usage
* Shows memory usage
* Lists top 5 running processes
* Saves output to the `logs/` folder

---

### 2. file_organizer.sh

This script organizes files based on file extensions.

Main functions:

* Creates `Images/`, `Documents/`, and `Others/` folders
* Moves `.txt`, `.md`, and `.pdf` files into `Documents/`
* Moves `.csv` and `.conf` files into `Others/`
* Prints a summary of files moved

---

### 3. create_student_folders.sh

This script reads data from `student_list.txt` and automatically creates student submission folders based on group number and matric number.

Main functions:

* Reads student data from `student_list.txt`
* Creates `student_submissions/` folder
* Creates group folders such as `Group_1`, `Group_2`, and others
* Creates matric number folders inside each group folder
* Skips empty or malformed lines
* Handles missing input file properly
* Saves results into a log file

## Packages Installed

The following packages were installed and tested:

```bash

```

| Package    | Purpose                             |
| ---------- | ----------------------------------- |
| `htop`     | Displays real-time system processes |
| `tree`     | Displays directory structures       |
| `neofetch` | Displays system information summary |
| `git`      | Used for version control            |
| `curl`     | Transfers data from URLs            |
| `wget`     | Downloads files from the internet   |

## How to Run the Scripts

Go to the scripts folder:

```bash
cd ~/linux_assignment/scripts
```

Give execute permission to the scripts:

```bash
chmod +x system_info.sh
chmod +x file_organizer.sh
chmod +x create_student_folders.sh
```

Run the scripts:

```bash
./system_info.sh
./file_organizer.sh
./create_student_folders.sh
```
